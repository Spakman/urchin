urchin by Spakman

A Unix shell for Ruby programmers.

Dependencies

A readline enabled Ruby (tested on Linux with MRI 1.8.7 and MRI 1.9.2) ruby-termios (tested with version 0.9.6)

Install

Installation is through Git just now. Just clone, list the tags and grab the latest release: git clone git://github.com/Spakman/urchin.git git tag -l git checkout 0.1.0
