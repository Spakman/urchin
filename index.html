Urchin

A Unix shell for Ruby programmers.

Dependencies

  • A Readline enabled Ruby (tested on Linux with MRI 1.8.7 and MRI 1.9.2)
  • ruby-termios (tested with version 0.9.6)

Install

Installation is best through Git just now. Just clone, list the tags and checkout the latest release:

  1. git clone git://github.com/Spakman/urchin.git
  2. git tag -l
  3. git checkout 0.1.0
